漫畫搜尋工具

前往安裝
Intro
透過輸入或鼠標選取的文字來搜尋 N 網漫畫，並且含有各種實用的快捷鍵 !!!
緣起

時常在一些留言彈幕中開到一串數字，雖然知道如何使用那串數字尋找漫畫，但是實在是太浪費時間了，少去了許多複製貼上以及輸入網址的時間。

特色
  • 隨時隨地使用
    為瀏覽器擴充套件，可直接在任何頁面上輸入號碼或關鍵字進行搜尋。
  • 免刪除歷程紀錄
    搜尋結果頁面為無痕分頁，再也不用刪除歷程紀錄。
  • 多個快捷鍵
    內建多個快捷鍵，快速前往熱門頁面。
  • 操作方便快速
    可用滑鼠選取頁面上的號碼或關鍵字進行搜尋，無需開啟擴充套件進行文字輸入。
聲明

這個擴充套件處經過多次審核皆無法通過，及無法上架至 Chrome 線上應用程式商店。
不過經過我多次上傳，並且在沒有修改任何程式碼的前提之下，終於審核通過並且上架至 Chrome 線上應用程式商店。
